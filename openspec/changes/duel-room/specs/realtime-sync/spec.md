## ADDED Requirements

### Requirement: User can generate sync code
The system SHALL allow a user to click "Link Device", generating a QR code or a short text link to their active duel session.

#### Scenario: Creating a room
- **WHEN** the user clicks "Link Device"
- **THEN** a modal appears with a 6-character room code and a scannable QR code

### Requirement: User can join a sync room
The system SHALL allow a second user to scan the QR code or enter the 6-character text link to join an active session.

#### Scenario: Joining a room
- **WHEN** user 2 scans the QR code generated by user 1
- **THEN** user 2 is immediately entered into the `/duel` route and their LP state perfectly matches user 1

### Requirement: State remains perfectly synced
The system SHALL synchronize any changes (LP, Timer, Logs, RNG) between all connected devices in a room.

#### Scenario: Updating synced state
- **WHEN** user 1 subtracts 1000 LP from Player 1
- **THEN** user 2's device instantly reflects Player 1 at 7000 LP and the Combat Log entry appears
