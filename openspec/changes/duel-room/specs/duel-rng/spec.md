## ADDED Requirements

### Requirement: User can flip a coin
The system SHALL provide an interface to randomly generate "Heads" or "Tails".

#### Scenario: Flipping a coin
- **WHEN** the user taps the coin flip tool
- **THEN** an animation plays, a result is generated, and a message is added to the Combat Log

### Requirement: User can roll a dice
The system SHALL provide an interface to randomly generate a number between 1 and 6.

#### Scenario: Rolling a dice
- **WHEN** the user taps the dice roll tool
- **THEN** an animation plays, a result 1-6 is generated, and a message is added to the Combat Log
