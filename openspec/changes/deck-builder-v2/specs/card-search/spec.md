## MODIFIED Requirements

### Requirement: User can filter search results
The system SHALL provide an interface to filter cards by type, attribute, level, and text, organized in a way that minimizes visual clutter on mobile.

#### Scenario: Using categorized filters
- **WHEN** the user opens the filter section
- **THEN** they see an organized, accessible layout (e.g., collapsible sections or pills) rather than all dropdowns visible at once
